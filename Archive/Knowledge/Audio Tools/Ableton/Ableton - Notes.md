# Ableton's ALC Files

To save samples as ALC ([Live Clip](https://www.google.com/search?client=safari&rls=en&q=Live+Clip&ie=UTF-8&oe=UTF-8&mstk=AUtExfDaLHuxGkQLbzDFs10-y9Gxb1MYZZqpkg8IdfeLEZViAnK41UaFPNs-oMdo9PiaBJCqB0yzHk08CYfIQXtmgNIk2arfNNWEVSQlQ75qYG9BoaFMnkDqweY3AYXN5UF5JXgH9OEmJgyhQ5uDbff-8V_g891QvBZBQEq6ZBrdbQ4DPAbc11vMANDt7-ziPis-SY9ODotsHXNAJixGhKz9uPxoDES6pDcvxYIi7_ij02o9qXwsY2cd3JPv2jCJ3sKsQV9Kth4pis0ai7JnzNO_l4eQ5wAtuyFOmNNJ6ujAHbtpjQ&csui=3&ved=2ahUKEwil9-birYqQAxVUnK8BHWxqAx0QgK4QegQIARAD)) files in your [User Library](https://www.google.com/search?client=safari&rls=en&q=User+Library&ie=UTF-8&oe=UTF-8&mstk=AUtExfDaLHuxGkQLbzDFs10-y9Gxb1MYZZqpkg8IdfeLEZViAnK41UaFPNs-oMdo9PiaBJCqB0yzHk08CYfIQXtmgNIk2arfNNWEVSQlQ75qYG9BoaFMnkDqweY3AYXN5UF5JXgH9OEmJgyhQ5uDbff-8V_g891QvBZBQEq6ZBrdbQ4DPAbc11vMANDt7-ziPis-SY9ODotsHXNAJixGhKz9uPxoDES6pDcvxYIi7_ij02o9qXwsY2cd3JPv2jCJ3sKsQV9Kth4pis0ai7JnzNO_l4eQ5wAtuyFOmNNJ6ujAHbtpjQ&csui=3&ved=2ahUKEwil9-birYqQAxVUnK8BHWxqAx0QgK4QegQIARAE), drag the desired audio or MIDI clip directly from your project's Session or Arrangement View into a folder within your User Library in the Ableton Browser. Ableton will then create the .alc file, automatically collecting and referencing any associated samples in a dedicated `"Samples/Imported"` folder within your User Library. For audio clips, this process uses your "Browser Behaviour" preference, so ensure "Collect Files on Export" is set to "Always" or "Ask" to copy the sample into the User Library.

## Analysis Files
- An analysis file can also store default clip settings for the sample:
- Clicking the Clip View’s [Save button](https://www.ableton.com/en/live-manual/11/clip-view/#saving-default-audio-clip-settings-with-the-sample) will store the current clip’s settings with the sample’s analysis file. The next time the sample is dragged into Live, it will appear with all its clip settings intact. This is particularly useful for retaining Warp Marker settings with the sample. Storing default clip settings with the analysis file is different from saving the clip as a Live Clip.
