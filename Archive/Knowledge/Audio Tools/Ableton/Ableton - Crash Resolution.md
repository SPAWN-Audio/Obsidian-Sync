> [!example] #### Finding Ableton Crash Issue
> On Mac if you open the crash report from the right panel when you relaunch Ableton, look for the crashed thread number (it’s somewhere near the top of the report). Scroll to the corresponding thread number and it will tell you the reason for the crash (eg what plugin is causing it). This is how I identified VSTs which crashed Ableton 11 but 12 has been rock solid and don’t know if you can see the crash report as before but hope this can help you.
