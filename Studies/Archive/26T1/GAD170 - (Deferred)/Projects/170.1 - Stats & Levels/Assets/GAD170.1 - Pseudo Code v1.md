# Pseudo Code v01
```
- Initialise Game
  - Initialise User Interface

- Reset Game
  - Load Player Stats - LVL 1

- New Round
  - Set Player HP to Max
  - Randomise Enemy stats
    - Load Enemy into Game
  - Player Attacks
    - attackStrength - Enemy HP

  - If Enemy HP = 0
    - Display: Congrats Enemy Defeated
    - Randomise XP Amount
    - Add to total Player XP
    - If, XP total = over Threshold
      - Display: LVL Up Ready, Press [SPACE] to evolve
      - If, Player Level = 5
        - Display: "Bool-Basaur Evolved to Struct-a-saur"
        - Game Won.
      - Else, 
        - Increase Attk (x 1.25) 
        - Increase XP Threshold
        - Start New Round
    - Else,
      - Enemy Attacks - based on Enemy's 'attackStrength'
      - If, player Health = 0
        - Display: "GAME OVER'
        - Reset Game
      - Else,
```
