## Debugging & Operations
You've learned about variables and operands, and now it's time to learn how to make sure you're using them correctly. 
In this next video, you'll learn about debugging, which is a key skill for making sure your games are glitch-free, and you don't waste your time trying to find bugs the hard way through trial and error. 

> *Learn these skills now, and they'll save you so much time later on, trust us!* 

Operators in Unity are essentially the building blocks of mathematical operations in your game's code. 
They include numbers, variables, or values that you use to perform calculations and make decisions within your game. 
Whether you're calculating the position of an object, determining damage dealt, or controlling character movement, operands are the elements you manipulate to achieve the desired outcomes.

## Scope & Local Variables
Local variables are like the secret notes you jot down during a quest, meant only for that specific part of an adventure or level. 
They're temporary and vanish once that part is over. On the other hand, global variables are the epic tales that spread throughout your entire game world. 

They're accessible from any corner of your game and stick around as long as your adventure continues. Watch the following video to learn more about which type of variable you should use when. [W02 Pre Class Video - Scope & Local Variables](https://youtu.be/cgcISLhiaV0)

### Logical Operators
- `&&` - Returns true if both boolean operands are true
- `||` - Returns true if at least one boolean operands is true
- `!` - Negates the boolean value of an operand

> **NOTE:**
> You can combine any of these together *ie*: "1 < 3|| 6 > 4 && 7 <9"

![[GAD170 - Logical Operators.png|567]]


### Input 
Input handling is crucial for creating interactive games. Unity provides the Input class to manage player input, such as keyboard, mouse, and touch events. Input handling enables players to control characters, trigger actions, and interact with the game world.

- **Note:** there are two input systems inside of Unity we'll be covering the New Input System
- **Note:** the new input system will need to have the library Unity Engine. InputSystem; added to your script.
