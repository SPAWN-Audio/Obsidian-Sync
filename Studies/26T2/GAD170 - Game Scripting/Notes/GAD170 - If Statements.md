# Notes

- `>=` - is like saying "is the value of score greater than or equal to 100?"
- `<=` - is like saying "is the value of health less than or equal to 0?"
- `==` - is like saying "is the value of coins equal to exactly 0?"
	- *in string checking - it's case sensitive*
- `!=` - is like saying "is the value of coins not equal to exactly 0?"
	- *think of it as **'NOT'** ie: `if(hasKey) != true` is like saying  if(hasKey is NOT true*
